## 负载均衡算法
### 随机法
未加权重（适合用服务器性能相近的公司），所有服务器被请求到的概率是一样的。加了权重可以避免这种情况
存在问题：有服务器可能永远也不会被请求到

### 轮询法
按个服务器来处理。可以配置加权，每个请求按照时间顺序逐一分配到不同的服务器处理
